display 속성에 대해 알아보자

  • display 속성은 화면에 어떻게 보여질지 결정하는 속성
  • html 태그는 기본적으로 block과 inline요소 둘 중에 하나를 기본 값으로 가짐

inline

  1. span, a, img 태그와 같이 내용물(content) 만큼만 공간이 할당됨
  2. content만큼 공간을 할당하기 때문에 width, height, padding, margin 사용불가
  3. 줄바꿈 없이 바로 옆에 배치
d-inline1 클래스를 갖고 있는 div
d-inline2

block

  1. div, p, ul, ol태그와 같이 한 줄을 다 차지함
  2. inline 요소와 달리 width, height, padding, margin 사용가능
d-block1 클래스를 갖고 있는 div
d-block2

inline-block

  1. block요소처럼 크기를 지정하되
  2. inline 요소처럼 다른 내용과 같은 줄을 공유하고 싶은 경우
  3. block과 inline요소의 장점을 가짐
